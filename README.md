# Behavioural Interview Simulator

Simple AI-driven behavioural interview simulator for practicing STAR-style answers. Built for a hackathon.

Run locally:

1. Install dependencies

```bash
npm install
```

2. Start server

```bash
npm start
```

3. Open http://localhost:3000 in your browser
Download the packaged project:

- If a ZIP was created, you can download it from the running server at `http://localhost:3000/download`.

Alternatively, the ZIP `behavioural-interview-simulator.zip` may be present in the project root after packaging.
